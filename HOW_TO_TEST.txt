===============================================================
TROUBLESHOOTING:
===============================================================

If nothing happens:
- Make sure you're in the oopscp folder
- Right-click oopscp.exe → "Run as administrator"
- Open PowerShell and type: .\oopscp.exe

If you see errors:
- Make sure all files are in the same folder
- Check that oopscp.exe exists
- Try rebuilding: run build.bat
